# Note Online 線上學習筆記
使用Angular4打造的網站。<br/>
1. 採用分層式架構: <br/>
Domain層:　NoteOnlineCore <br/>
Repository層:　NotesOnlineRepository <br/>
Service層:　NotesOnlineService <br/>
UI層:　NotesOnlineWebApi、NotesOnlineAngular4 <br/>
